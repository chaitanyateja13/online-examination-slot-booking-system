# Maintenance Guide

## Routine Maintenance

- Keep project dependencies updated.
- Regularly back up the MySQL database.
- Monitor application logs.
- Remove unused files and temporary data.

---

## Security Maintenance

- Update secret keys when necessary.
- Review user roles and permissions.
- Install security updates for dependencies.

---

## Database Maintenance

- Optimize database queries.
- Check database integrity.
- Archive old examination records when appropriate.

---

## Performance

- Optimize static assets.
- Monitor server performance.
- Review slow database queries.

---

## Backup Strategy

- Daily database backups.
- Weekly project backups.
- Verify backup restoration periodically.
