# Testing Guide

## Objective

This guide outlines the manual testing process for the Online Examination Slot Booking System.

---

## Student Module

- Login using valid credentials.
- Book an examination slot.
- Download hall ticket.
- View examination results.
- View notifications.

---

## Faculty Module

- Login successfully.
- View student bookings.
- Publish examination results.
- Send notifications.

---

## Payment Module

- Verify payment status.
- Confirm successful transaction.
- Validate payment records.

---

## Database Testing

- Verify database connectivity.
- Check CRUD operations.
- Ensure referential integrity.

---

## Browser Compatibility

- Google Chrome
- Microsoft Edge
- Mozilla Firefox
